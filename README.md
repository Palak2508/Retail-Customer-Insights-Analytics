# 📖 Project Overview

Retail businesses generate large volumes of customer transaction data every day. Converting this raw data into meaningful insights helps organizations understand customer behavior, improve marketing strategies, and make informed business decisions.

This project focuses on analyzing retail customer purchase data through data cleaning, SQL-based analysis, and interactive dashboard development. The analysis highlights customer spending patterns, subscription trends, product performance, and key business metrics.

---

# 🎯 Project Objectives

- Analyze customer purchasing behavior
- Identify high-value customer segments
- Compare spending patterns across different customer groups
- Evaluate subscription and discount impact
- Analyze product category performance
- Build an interactive Power BI dashboard for business reporting

# 📂 Dataset

The dataset contains approximately **3,900 retail customer transactions** with information including:

- Customer Demographics
- Product Categories
- Purchase Amount
- Subscription Status
- Shipping Type
- Previous Purchases
- Discounts Applied
- Customer Ratings
- Purchase Frequency
